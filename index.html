Open Weather Map

Datenlizenz Deutschland - Version 2.0

Interaktive Karte

Interaktive Karte mit den Wetterdaten des Deutschen Wetterdienstes auf Basis der Stations Kennungen.

Datenquelle

Die Liste der Wetterstationen und deren Daten werden vom Deutschen Wetterdienst auf folgender Seite https://opendata.dwd.de zum Download angeboten. Für die Abfragen der einzelnen Datensätze werden die Endpunkte der API kann https://dwd.api.bund.dev aufgerufen werden.
